Over mijzelf

Wie ben ik

Ik ben Kornel Kujawa, ik ben 19 jaar oud en ik ben een individu uit sort homo sapiens ook wel als moderene mens bekend(Ik zou echter een luiaard moeten zijn). Ik kom uit Polen een land op de planeet aarde, maar in 2013 heb zich verhuisd naar Nederland ook een land op aarde. Rond 2015 begon ik zich leren programmeren in C# met behulp van Unity Engine. Heb geen echte dingen ermee gedaan, meer simple experimenten meestal in gebied van procedurele generatie. Op dat moment heb ik een kleine project met procedurele animatie die ik zou graag ooit in een spel maken. Op dat moment studeer ik in de opleiding Creative Media and Game Technologien, ervor heb ik VWO afgerond. Ik hoop dat ik zou de rest van mijn leven achter de computer scherm kunnen leiden.

Mijn hobby's

Van de wie ben ik sectie is het makkelijk te afleiden dat ik hou van computer's en programmeren hou. Behalve programmeren spel ik ook vak computer spelletje's. Ik hou ook van andere organismen zoals honden, katten, ratten met de uitzondering van andere mensen en somige wat meer gevarlijke bacterieen en virusen. Een andere ding die ik fijn vind is slapen.

Wat kan ik

Op dat moment beschiek ik over volgende vaardigeheden:

  • gebruik van Unity Engine

  • programmeren in C#

  • basic 3d modeleren in Blender

  • basic teken in Krita

  • basic programmeren in javascript

  • basic programmeren in html en css

Wat wil ik leren

Ik zou graag willen mijn computer kennis vergroten, voornamelijk in de richting van computer game's, verder zou ik graag ooit op een muziek instrument willen leren maar dat komt pas nadat ik mijn eerste leerdoel wat meer haalt. Ook een andere ding dat ik wil verbeteren is mijn kennis van Nederlandse taal zodat ik beter kan communiceren met anderen.

Het is alleen proef dat ik kan images op mijn site zetten, omdat ik had geen idee wat zou ik in de zijkant voor kan gebruiken, gebruik ik dat tijdelijk voor een beeld van pizza en uitleg waarom er aan de zijkant een beeld van pizza zich bevindt.
